Словарь представляет собой словарь в словаре.
Первым ключом выступает само слово, например 'key'.
Вторым ключом выступает категория связанных слов, указанные в abbreviation.txt

Пример:
eng_dict['key']['n'][2]
где,
eng_dict - словарь
['key'] - слово
['n'] - связанная с ним категория (в данном случае категория сущ.)
[2] - третье слово в этом списке (счет начинается с 0)

Файлы
add_word.py - обрабатывает данные из source.txt и добавляет в словарь eng_dict.json
dict_3name_check.py - проводит анализ по трем заданным словам и выдает результат в формате txt.
