Словарь и метод создан в целях сравнения коллокаций различных слов. 
Текущая версия 3.7

На определенном этапе изучения иностранного языка учащемуся важно сравнивать между собой синонимы, и одним из способов их сравнения - сопоставлять с какими словами они чаще всего употребляются, то есть коллокации. 

Структура:
Первым ключом выступает само слово, например 'key'.
Вторым ключом выступает категория слова, указанная в abbreviation.txt
Третьим ключом выступает 'collocations' и 'meaning' 
Четвертым ключом в 'collocations' выступает категория связанных слов, указанные в abbreviation.txt
Четвертым ключом в 'meaning' является значение слова.
Пятым ключом выступает множество слов (в 'collocations').

Пример:
eng_dict['key']['adj]['collocations']['n']
где,
eng_dict - словарь
['key'] - слово
['adj'] - категория слова
['collocations']
['n'] - связанная с ним категория (в данном случае категория сущ.)
Выдаст множество слов - их нельзя перебрать при помощи индексов

eng_dict['key']['adj]['meaning']
где,
eng_dict - словарь
['key'] - слово
['adj'] - категория слова
['meaning']
Выдаст значение слова

Файлы
add_word.py - обрабатывает данные из source.txt и добавляет в словарь eng_dict.pickle
words_comparing.py - проводит анализ по трем заданным словам и выдает результат в формате txt.
eng_dict.pickle - словарь
test.py - вспомогательный файл
important - вспомогательный бинарный файл 
source.txt -  текстовый файл, куда можно вписать список слов для последующего добавления в словарь в качестве коллокаций к слову. 
results_...txt - файл, содержащий результат запроса

Папка results - место хранение текстовых файлов, содержащих результат сравнения слов
